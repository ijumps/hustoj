# 系统安装软硬件需求 #




# 安装版 #

最低配置:
  * PIII-450
  * 128M
  * 10G
  * 10/100M
推荐配置:
  * CPU 1G+ x2
  * 1G+ RAM
  * 100G+ HD
  * 100/1000M LAN
软件系统：
  * GNU/Linux 32bit (推荐Ubuntu9.04+)
  * GCC 4.3.0+
  * MySQL 5.0+
  * Apache2.0+
  * PHP   5.3+


# LiveCD版 #

最低配置:
  * K6-2-450
  * 256M
  * CDROM
  * 无须硬盘
  * 无网络可本地运行、有网络可升级至最新版
